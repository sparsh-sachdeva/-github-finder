# 🔍 GitHub Finder

A simple and clean web app to search GitHub users and view their profile details along with their top repositories.

---

## 🚀 Features

- 🔎 Search GitHub users by username  
- 👤 View profile details:
  - Name
  - Username
  - Bio
  - Followers / Following
  - Public repositories count  
- 📦 View top 5 repositories  
- 🔗 Direct link to GitHub profile and repositories  
- ⚠️ Error handling:
  - Invalid input
  - User not found (404)
  - Network errors  
- 🚫 Prevents multiple requests (button disable)

---

## 🛠️ Tech Stack

- HTML  
- CSS  
- JavaScript (Vanilla JS)  
- GitHub REST API
